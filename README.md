A modern, responsive personal portfolio website showcasing my academic and professional achievements in Data Science, AI, and Education Technology.

Visit: **https://mahmoud-abujadallah.github.io/mahmoud_abujadallah/**

> Update the URL above with your actual GitHub username.

## Features

- Dark theme with modern UI design
- Typewriter hero animation
- Scroll-reveal animations
- Animated counters
- Responsive layout (mobile, tablet, desktop)
- Sections: About, Skills, Experience, Education, Publications, Contact

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts (Inter, JetBrains Mono)
- Font Awesome Icons
- GitHub Pages for hosting

## Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set Source to **Deploy from a branch**
4. Select **main** branch and **/ (root)** folder
5. Click Save — your site will be live in ~1 minute
